# 安装与删除包

## 安装包
##### 全局
```
npm install <name> --global		# 全称
npm install <name> -g			# 简写
```
##### 本地项目
```
npm install <name>
```
##### 配置信息 - 项目依赖
```
npm install <name> --save
```
##### 配置信息 - 项目开发环境
```
npm install <name> --save-dev
```

## 删除包
##### 全局
```
npm uninstall <name> --global  	# 全称
npm uninstall <name> -g			# 简写
```
##### 本地项目
```
npm uninstall <name>
```
