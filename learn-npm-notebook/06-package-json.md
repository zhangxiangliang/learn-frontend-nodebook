# 配置文件 - package.json

## 初始化配置文件
命令
```
npm init
```

## 配置文件信息
```
name				# 项目名字，默认是 当前目录的名字
version			 	# 版本号，默认是 1.0.0
description:		# 项目的描述
entry ponit			# 项目的入口，默认是 index.js
test command		# 测试项目的命令
git repository		# 项目的 git 仓库地址
keywords			# 项目的关键词，便于他人搜索项目
author				# 项目的作者
license				# 项目的授权方式
dependencies		# 项目依赖的包
devDependencies		# 项目开发环境的包
```
