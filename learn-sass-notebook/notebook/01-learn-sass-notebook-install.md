# Sass 安装
## 系统环境
因为 `Sass` 是基于 `Ruby` 开发的，所以得先安装 `Ruby`。
## 安装
安装`Sass`最新版本
```
gem install sass
```
## 常用命令
##### 更新 `sass` 版本
```
gem update sass
```
##### 查看 `sass` 版本
```
sass -v
```
##### 查看 `sass` 帮助
```
sass -h
```
##### 编译 `sass` 文件
```
sass name.scss:name.css
// 或者
sass scss-filename/name.scss:css-filename/name.css
```
##### 自动编译 `sass`
```
sass --watch scss-filename:css-filename
```
